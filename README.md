# DiseaseVisualiser

> Inspired by the onset of COVID-19 and the challenges it brings, this is a fully customisable tool that helps visualise how a contageous virus moves through a population.

---

# Usage

## Hotkeys
* `SPACE` = change view mode
* `ESC` = exit
